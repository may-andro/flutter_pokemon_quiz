## Purpose

This modules assemble and expose the set of data in form of repositories.

## Setup

The module exposes the ```main.dart``` with a function ```setUpDataModule``` for initializing the module.

## Usage

```yaml
dependencies:
  data:
    path: ../data
```

## Dependencies

```yaml
dependencies:
  dependencies:
    path: ../dependencies
  local:
    path: ../local
  network:
    path: ../network
```
